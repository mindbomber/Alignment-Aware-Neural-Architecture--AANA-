# Assets

This folder contains public-facing project images.

## GitHub social preview

Use `github-social-preview.png` as the repository social preview image.

To set it on GitHub:

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Scroll to **Social preview**.
4. Upload `assets/github-social-preview.png`.
5. Save changes.

Recommended GitHub social preview size is 1280 x 640 pixels. This image is already sized to that format.
