# Diagnostic Probes

This folder contains probe-only benchmark diagnostics. These files may include benchmark literals, hand-discovered recovery flows, or narrow failure-mode probes.

Probe code is not part of the public examples surface and must not support public AANA performance claims. General architecture examples belong under `examples/`; benchmark probes belong here.

Probe runs must remain opt-in and clearly labeled `diagnostic`, `probe`, or `diagnostic_probe_only`.
