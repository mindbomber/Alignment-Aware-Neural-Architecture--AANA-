# Alignment as a Dynamical System Under Incomplete Constraint Representation

This is a clean Overleaf/arXiv-ready package.

## Compile
Use `pdflatex main.tex` twice.

The paper uses native TikZ/PGFPlots figures directly inside `main.tex`; no external image files are required.

Included files:
- `main.tex`: full paper source
- `references.bib`: optional BibTeX source retained for future editing
- `README.md`: this file

Note: The results section has been updated with the 40-prompt, 160-output pilot results. The pilot is presented as a directional first test, not as a final benchmark claim.
