# ATS/AANA NeurIPS-Style Overleaf Project

This project is a standalone pdflatex-compatible arXiv/NeurIPS-style paper with native TikZ/PGFPlots figures.

## Compile
Upload the folder to Overleaf and compile `main.tex` with `pdfLaTeX`.

Locally:

```bash
pdflatex main.tex
pdflatex main.tex
```

## Files
- `main.tex`: full standalone paper.
- `README.md`: compile instructions.
- `SUBMISSION_CHECKLIST.md`: NeurIPS-style submission checklist.
