# Alignment-Aware Neural Architectures Overleaf Source

This folder contains editable LaTeX source for the AANA framework manuscript.

## Compile

Use:

```powershell
pdflatex main.tex
pdflatex main.tex
```

The source uses native TikZ/PGFPlots figures and does not require external image files.

Included files:
- `main.tex`: full paper source
- `README.md`: this file

Note: The results section reports the repository's current 120-row matched constraint-reasoning pilot. It is model-judged pilot evidence, not a final benchmark claim.
